# PetC acquires Vet — Post-Merger Integration Plan

**Case Study | March 2025**

A comprehensive Post-Merger Integration (PMI) strategy for PetCare's acquisition of VetWell, 
covering business model redesign, synergy quantification, operating model, governance structure, 
and change management plan.

## 🔍 Key Highlights

- **Business Model:** Tiered dual-brand strategy combining PetCare's scale with VetWell's premium services
- **Synergies:** €21M total value unlocked — €9.9M revenue growth + €11.5M cost savings
- **EBITDA Impact:** From €5.2M (11% margin) to €17.5M (30% margin) by 2027
- **Integration Roadmap:** 5 initiatives prioritized by impact/effort across brand, IT, procurement, HR, and workforce
- **Governance:** Steering Committee → Integration Coordinator → PMO → workstream teams
